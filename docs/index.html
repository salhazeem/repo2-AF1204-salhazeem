Projects

📈 Project 1 — Stock Price Analysis: Apple vs Tesla (2020–2024)

Using real daily closing prices downloaded from Yahoo Finance via yfinance, this project compares how Apple (AAPL) and Tesla (TSLA) performed over five years. Apple grew steadily and consistently, while Tesla experienced sharp rises and falls — a clear sign of higher risk. The line chart makes it easy to spot key turning points, such as Tesla's dramatic rise in 2020 and its steep correction in 2022.

💡 Key insight: Despite Tesla's higher total return of +1355.2%, its price swings were far more extreme — making Apple the more stable long-term investment over this period.

🔧 Created using Python — yfinance (real data), pandas, matplotlib

📅 Project 2 — Monthly Return Comparison: Apple vs Tesla (2023)

This project calculates the month-by-month percentage returns for Apple and Tesla throughout 2023 using pandas resampling and pct_change on real price data. The grouped bar chart reveals how each company performed in the same market conditions — Tesla's bars are consistently taller in both directions, confirming its much higher short-term volatility compared to Apple's steadier monthly performance.

💡 Key insight: Tesla's annualised volatility of 67.2% is more than double Apple's 31.7% — meaning investors in Tesla faced significantly more uncertainty for each unit of return.

🔧 Created using Python — yfinance (real data), pandas (resample, pct_change), matplotlib

📊 Project 3 — S&P 500 Sector Returns & Rolling Volatility (Interactive)

This project has two views you can switch between using the dropdown. The first shows how five major S&P 500 sectors performed in 2023 — Technology led with +56.4% while Energy was the only sector to fall. The second view shows 30-day rolling volatility for Apple and Tesla calculated from real daily returns, a standard risk measure used in finance to track how risk changes over time.

💡 Key insight: The 2020 COVID crash caused a sharp spike in volatility for both stocks, but Tesla's risk remained elevated for much longer — reflecting the uncertainty around its rapid growth phase.

🔧 Created using Python — pandas (rolling std, annualisation), matplotlib, real yfinance data
